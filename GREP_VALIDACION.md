## ✅ Pruebas Recomendadas

### Antes de producción:

```bash
# 1. Validar sintaxis
php -l grep.php
# Resultado esperado: ✅ No syntax errors

# 2. Test búsqueda simple
php -r "
require 'grep.php';
\$e = new TextSearchEngine(__DIR__);
\$r = \$e->search('php');
echo 'Resultados: ' . count(\$r) . "\n";
"

# 3. Test regex
php -r "
require 'grep.php';
\$e = new TextSearchEngine(__DIR__);
\$r = \$e->search('^class ', true, false, true);
echo 'Clases encontradas: ' . count(\$r) . "\n";
"

# 4. Test seguridad
php -r "
require 'grep.php';
try {
  \$e = new TextSearchEngine('/etc/');
  echo 'No debería llegar aquí\n';
} catch (Exception \$ex) {
  echo 'Validación correcta: ' . \$ex->getMessage() . "\n';
}
"
```
